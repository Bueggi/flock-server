## Installation
Clone the repo onto your local machine and run `npm i`.
In order to test the application you can run `npm run dev`.

Important:
Create a new `.env` file and include following properties with your own values:
```
FACEBOOK_APP_ID=''
FB_APP_SECRET={''}
FACEBOOK_CALLBACK=''
```
